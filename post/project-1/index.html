Project 1: Movie Genre Classifier

April 19, 2022
  • Final Year Undergraduate Project, worked in a team of four and coordinated the team in the project.
  • Created a bi-directional Long Short-Term Memory (LSTM) Model to classify into top five genres: thrillers, Crime Fiction, Romance, Action, and Short films.
  • The data used for the project was from the CMU Movie Summary Corpus (http://www.cs.cmu.edu/~ark/personas/) collected by David Bamman, Brendan O’Connor, and Noah Smith at the Language Technologies Institute and Machine Learning Department at Carnegie Mellon University.
  • Points of improvement:
    • Requires a robust system to run the model faster for better hyper-parameter selection.
    • Would recommend to use a different Recurrent Neural Network (RNN)-based model
