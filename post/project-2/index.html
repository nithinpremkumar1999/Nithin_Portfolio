Project 2: Predicting VIX Index

April 20, 2022
  • Final Year Undergraduate Project, worked individually to find an appropriate traditional Statistical Model to predict future VIX values.
  • The data set contains information from 1990, February 01 to 2021, December 13.
  • Followed the Box-Jenkins method to find the appropriate p, d, and q values for ARIMA model.
  • Points to improve:
    • Model was not able to predict the sudden fluctuations due to external factors.
    • Could have implemented a Deep Learning Model in addition to achieve a better result.
